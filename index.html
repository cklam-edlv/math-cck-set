🎉 集合小英雄 🎉

✨ 什麼是「集合」？

把具有相同特徵的東西放在一起，就是一個集合。

✅ 屬於：如果東西在集合裡，就是「屬於」這個集合。

📌 例如：{ 🍎, 🍌, 🍓 } 就是「水果集合」。

🌟 恭喜你完成了基礎集合概念的挑戰！

📝 列舉法與描述法

✨ 列舉法：將所有元素列出，如 { 🍎, 🍌, 🍓 }。

✨ 描述法：用條件描述，如 { x | x 是水果 }。

🌟 你已能靈活轉換列舉法與描述法！

🔗 交集 (Intersection)

✨ 交集 是指兩個集合中共同擁有的元素。

📖 記作 A ∩ B，意思是「同時屬於A和B」的東西。

🌟 例如：集合A = { 紅色物品 }，集合B = { 水果 }
交集 = { 紅色水果 } 如 🍎 蘋果、🍓 草莓。

🧩 併集 (Union)

✨ 併集 是把兩個集合的所有元素合在一起。

📖 記作 A ∪ B，包含所有屬於A或屬於B的元素。

🌟 例如：集合A = { 貓, 狗 }，集合B = { 狗, 兔子 }
併集 = { 貓, 狗, 兔子 }。

📂 子集 (Subset)

✨ 子集：如果集合 A 的所有元素都屬於集合 B，則 A 是 B 的子集。

📖 符號記作 A ⊆ B（A 包含於 B）。

🌟 你已經學會判斷子集！接下來挑戰「真子集」分辨遊戲吧！

📂 真子集 (Proper Subset)

✨ 真子集：如果 A ⊆ B 且 A ≠ B，則 A 是 B 的真子集。

📖 符號記作 A ⊂ B。

🌟 例如：{ 紅色水果 } ⊂ { 水果 }
{ 水果 } ⊆ { 水果 }（相等，所以不是真子集）

🌍 全集與差集

✨ 全集 U：所有被考慮的對象。

✨ 差集 A - B：在 A 中但不在 B 中的元素。

🌟 你已經學會了全集與差集！
